# Chilete DevPath Website

[Spanish version](README.es.md)

Personal website for Chilete DevPath. This repository contains the public version of my portfolio, focused on presenting professional identity, learning path, projects, and technical growth.

## Purpose

To work as the main showcase for my personal brand. Unlike `learning-labs` and `academic-projects`, this repository does not store exercises or complete academic deliverables: it presents a curated web experience that explains who I am, what I am building, and how my developer profile is evolving.

## Contents

- `index.html`: main website page.
- `css`: styles organized by base, components, layout, pages, and themes.
- `js`: interactive behavior for the site.
- `assets`: images, icons, and visual resources.
- `manifest.webmanifest`: basic PWA configuration.

## Technical Focus

- Semantic HTML.
- Modular and responsive CSS.
- JavaScript interactions.
- Structure prepared to grow with new projects.
- Visual identity aligned with Chilete DevPath.

## Relationship With Other Repositories

- `learning-labs`: evidence of progressive technical learning.
- `academic-projects`: academic projects and formal deliverables.
